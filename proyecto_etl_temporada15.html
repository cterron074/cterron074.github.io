Pipeline ETL: Estadísticas de la Temporada 15

Automatización de extracción, transformación y almacenamiento de métricas de rendimiento deportivo.

Descripción del Proyecto

Este proyecto implementa una infraestructura completa de datos diseñada para capturar, procesar y consolidar las estadísticas complejas de la Temporada 15. El objetivo principal es transformar datos brutos provenientes de múltiples fuentes en un repositorio estructurado óptimo para el análisis analítico y de rendimiento.

Nota de desarrollo: La interfaz interactiva del dashboard se encuentra actualmente en fase de despliegue final. Puede explorar la arquitectura técnica abajo o interactuar con el contenedor una vez esté operativo.

Tecnologías Utilizadas

Python Pandas SQLAlchemy Flask API

Arquitectura de Datos (ETL)

1. Extracción Consumo de archivos estructurados de League of Legends (data.csv).
2. Transformación Limpieza de nulos, estructuración del modelo y normalización de métricas.
3. Carga Inserción directa y optimizada en tu base de datos en la nube de Aiven.
